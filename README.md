# Acoustic-Based-Machine-Health-Monitoring
This project presents an **acoustic-based machine health monitoring system** designed to detect abnormal machine behavior using sound signals. Machine audio is collected and processed by converting it into **Mel-spectrogram representations**, which capture the frequency characteristics of the sound. A **Convolutional Autoencoder (CAE)** is then trained using only normal machine operation data to learn typical acoustic patterns. During testing, the model reconstructs the input spectrogram, and the **reconstruction error** is used as an indicator of abnormality. If the error exceeds a predefined threshold, the system identifies the sound as abnormal. This approach enables **non-invasive and cost-effective fault detection**, making it suitable for predictive maintenance and industrial machine monitoring applications.
